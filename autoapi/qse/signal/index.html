qse.signal#

This module contains Signal ans Signals classes to define a a quantity as a function of time. It can be used to express quantities like time dependent external field coupled to Hamiltonian.

class qse.signal.Signal(values, duration=None)[source]#

The Signal class represents a 1D signal with values and duration. The duration must be divisble by the number of values and each value is assume to last for an equal duration.

Parameters:

  • values (list | np.ndarray) – The values of the signal.

  • duration (int) – Duration of the signal. Defaults to the length of the passed values.

Examples

To create a constant signal, passing a single value:

import qse
s = qse.Signal([1], 10)
print(s)

Signal(duration=10, values=[1.])

To create an arbitrary signal, pass an array whose length is equal to the duration:

import qse
import numpy as np

ss = qse.Signal(np.linspace(0, 1, 5), 5)
print(ss)

Signal(duration=5, values=[0.   0.25 0.5  0.75 1.  ])

Arithmetic operations with scalars is supported. Adding or multiplying a scalar to a Signal returns a new Signal with modified values and the same duration. For example:

import qse
signal = qse.Signal([1, 1])
signal = signal * 3 + 0.5
print(signal)

Signal(duration=2, values=[3.5 3.5])

class qse.signal.Signals(signals=None)[source]#

The Signal class represents a collection of Signals.

Parameters:

signals (list[qse.Signal], optional) – The signals.

Examples

import qse
import numpy as np

x = qse.Signal([1], 10)
y = qse.Signal(np.linspace(0, 1, 5), 5)
ss = qse.Signals([x, y])
print(ss)
z = qse.Signals()
z += x
z += y
print(z)
